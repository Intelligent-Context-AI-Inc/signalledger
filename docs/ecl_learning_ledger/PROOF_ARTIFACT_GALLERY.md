# Proof Artifact Gallery

This gallery shows sanitized examples of the local artifacts generated by the ECL Learning Ledger GitHub Action and CLI.

The examples are safe for demos. They contain hashes, counts, controlled tags, and risk labels only. They do not contain dataset rows, prompts, completions, token sequences, embeddings, model weights, notebook cells, raw diffs, or secrets.

## Files

| Artifact | Example path | Purpose |
| --- | --- | --- |
| Append-only ledger | `examples/proof_artifact_gallery/events.jsonl` | Local history of metadata-only events |
| Risk report | `examples/proof_artifact_gallery/risk-report.md` | PR-friendly training-data risk summary |
| Compliance passport | `examples/proof_artifact_gallery/compliance-passport.md` | Audit/passport summary from ledger evidence |
| Verification file | `examples/proof_artifact_gallery/verification.json` | Hash-chain continuity result |
| PR comment | `examples/proof_artifact_gallery/pr-comment.md` | Sticky comment body for GitHub PRs |
| Auditor evidence manifest | `examples/proof_artifact_gallery/auditor-evidence-manifest.json` | Bundle index for security and audit review |

## Buyer Talking Point

ECL turns a training-data PR into a local evidence bundle. A reviewer does not need access to the dataset to see whether the metadata was clean, whether the risk gate fired, whether the compliance passport was generated, and whether the ledger remains verifiable.
