# Adoption Snapshots

This folder stores scheduled public adoption snapshots for SignalLedger.

The archive records GitHub repository traffic and public workflow-reference
search results before GitHub's traffic window expires. It is a directional
adoption signal, not an exact external run counter.

Snapshots must not contain datasets, prompts, completions, embeddings, token
sequences, model weights, raw diffs, secrets, private customer data, or private
repository contents.
